# به‌روزرسانی تقویم شمسی

این نسخه تقویم شمسی را به فرم‌های زیر اضافه می‌کند:

- ثبت هزینه
- ثبت قبض و تاریخ سررسید
- ثبت قسط و سررسید بعدی

تاریخ را می‌توان از تقویم انتخاب کرد یا با قالب `۱۴۰۵/۰۴/۲۷` نوشت. اعداد فارسی و انگلیسی هر دو پذیرفته می‌شوند.

اطلاعات نسخه قبلی حذف نمی‌شود؛ کلید ذخیره‌سازی برنامه تغییری نکرده است.

برای به‌روزرسانی GitHub Pages، همه فایل‌های این پوشه را در ریشه Repository جایگزین کن. بعد از سبزشدن عملیات Pages، اپ نصب‌شده را یک‌بار کامل ببند و دوباره باز کن.
