# روزمره‌یار — نسخه PWA اصلاح‌شده

برای نصب روی موبایل، این پوشه باید روی یک میزبان HTTPS منتشر شود. اجرای مستقیم فایل index.html یا لینک دانلود ChatGPT شرایط نصب PWA را فراهم نمی‌کند.

## انتشار سریع

- محتوای همین پوشه را روی Netlify، Vercel، GitHub Pages یا هر میزبان HTTPS قرار دهید.
- سپس آدرس سایت را مستقیماً در Chrome اندروید یا Safari/Chrome آیفون باز کنید.
- داخل تنظیمات اپ روی «نصب روزمره‌یار» بزنید یا از منوی مرورگر Add to Home screen / Install app را انتخاب کنید.

اطلاعات فعلاً در LocalStorage همان مرورگر و دستگاه ذخیره می‌شود.
